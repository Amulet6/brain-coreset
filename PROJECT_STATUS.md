## 一、最终实验结果 (CLIP ViT-B-32 视觉+文本, 时序堆叠, 3 次试验)

### 1.1 核心对比 (双头 MLP: 6-DoF 回归 + 夹爪分类)

```
┌──────────────────────────┬──────────┬──────────────┬─────────────┐
│ 方法                      │ MSE_6d   │ Gripper Acc  │ Gripper F1  │
├──────────────────────────┼──────────┼──────────────┼─────────────┤
│ Full 100%                 │ 0.772    │ 61.45%       │ 0.397       │
│ Random 10%                │ 0.863    │ 59.79%       │ 0.368       │
│ BRAIN-Coreset 10%         │ 0.822    │ 61.06%       │ 0.451       │
├──────────────────────────┼──────────┼──────────────┼─────────────┤
│ vs Random                 │ +4.7%    │ +1.27%       │ +22.6% ★    │
└──────────────────────────┴──────────┴──────────────┴─────────────┘
```

**结论：BRAIN-Coreset 在 6-DoF 回归上改善 4.7%，夹爪 F1 改善 22.6%。**

### 1.2 四个关键修复

| # | 问题 | 修复 | 效果 |
|---|------|------|------|
| 1 | 丢失语言模态 | CLIP Text Encoder 编码指令, 拼接到视觉特征 (2048d) | VLA 合规 |
| 2 | 夹爪 MSE 谬误 | 双头 MLP: 前6维回归(MSE) + 夹爪分类(BCE+Sigmoid) | F1 +22.6% |
| 3 | 5000帧内存借口 | 删除硬截断, 全量候选池 6998 帧, 核矩阵 187MB | 算法严谨 |
| 4 | 线性模型辩护 | 报告中正面回应: 线性模型的弱表达能力恰是其过滤优势 | 理论自洽 |

### 1.3 模型架构

```
输入: [v_{t-2}, v_{t-1}, v_t, lang] = 2048d
       └── CLIP 视觉时序 ──┘ └ CLIP 文本 ─┘

MLP: 共享层 2048→256→128
       ├─ 回归头 → 6-DoF 关节角度 (MSE Loss)
       └─ 分类头 → Sigmoid → 夹爪开/合 (BCE Loss)
```

### 1.4 迭代路线

```
v1  ResNet-18 + 高误差        512d     -6.0%   方向性错误
v2  ResNet-18 + 原型          512d     +0.5%   特征退化
v3  CLIP单帧  + 原型          512d     +3.1%   CLIP 解锁
v4  CLIP时序  + 原型          1536d    +7.1%   时序上下文
v5  CLIP VLA  + 夹爪分类      2048d    +4.7% (MSE_6d)
                                    +22.6% (Gripper F1)  ← 最终版
```
