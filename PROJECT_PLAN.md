# BRAIN-Coreset 实施方案

## 一、项目概述

### 1.1 题目

**题目2：基于脑启发核心集选择的轻量级 VLA 机械臂动作预测**

### 1.2 核心命题

> 借鉴人脑的预测编码与 RAS 机制，设计自动化数据修剪算法，从全量 ALOHA 数据中智能筛选 10% 核心集，在动作预测 MSE 上显著优于随机 10% 基线。

### 1.3 预期成果

- **量化目标**: BRAIN-Coreset 10% 的 MSE 相比 Random 10% 降低 **≥20%**
- **可视化**: t-SNE 证明核心集更均匀覆盖数据流形
- **认知深度**: 每个算法模块对应具体脑机制，有文献支撑
- **报告**: 7-8 页，≥7 张图表，图文并茂

## 二、算法设计

### 2.1 数据流 (Pipeline)

```
全量数据 (100%, ~25,000 帧)
        │
    ┌───┴───┐
    ▼       ▼
阶段1      阶段2
预测编码    RAS事件
    │       │
    └───┬───┘
        ▼
   候选池 (并集, 25% ± 2%)
        │
        ▼
   阶段3 Facility Location
   Lazy Greedy 子模优化
        │
        ▼
   核心集 (精确 10%)
        │
        ▼
   [可选] 阶段4 梯度精炼
   (加分项, 等量置换)
        │
        ▼
   MLP 训练 + MSE 评估
```

### 2.2 阶段1: 预测编码时序过滤

**对应脑机制**: Predictive Coding (Millidge et al., 2021)
— 大脑持续生成预测，只在预测误差大时激活

**算法**:
```
对每个 episode:
  1. 用轨迹的全局线性模型 W (R^{3×7} → R^7) 拟合
  2. 对每个 t ∈ [3, T]:
     x_t = [a_{t-3}, a_{t-2}, a_{t-1}]  # 拉平为 21d
     â_t = x_t · W
     e_t = ||a_t - â_t||_2
  3. 保留 e_t > μ + k·σ 的帧
```

**关键约束**:
- W 的拟合样本必须排除目标帧及其相邻帧 (至少隔离 1 帧)
- k 值通过二分搜索确定，使阶段1输出约为全量 35-40%

### 2.3 阶段2: RAS 关键事件检测

**对应脑机制**: Reticular Activating System
— 过滤背景噪音，聚焦高信息效用瞬间

**算法**:
```
对每个 episode:
  1. 夹爪突变帧: |gripper_velocity| > 90% 分位
  2. 加速度峰值帧: ||a''_t|| > 90% 分位
  3. 接触窗口帧: 速度下降率 > 80% 分位 AND 夹爪值开始变化
  标记为RAS事件帧
```

**关键约束**:
- 所有阈值基于全局统计分位数，不使用绝对数值
- 夹爪状态判定使用连续梯度，不使用布尔 0/1 判断

### 2.4 阶段3: Facility Location 子模优化

**对应脑机制**: 海马体模式分离 — 避免记忆高度相似重复经验

**算法**:
```
预处理:
  v_i: 512d ResNet-18 特征 → PCA → 32d → L2 normalize
  a_i: 7d 动作 → L2 normalize

相似度核 (乘积核):
  K_v(i,j) = exp(-||v̂_i - v̂_j||² / σ²_v)
  K_a(i,j) = exp(-||â_i - â_j||² / σ²_a)
  S(i,j) = K_v(i,j) · K_a(i,j)    # "与"关系

带宽估计:
  σ²_v = median({||v̂_i - v̂_j||²}) 在 5000 随机对上
  σ²_a = 同上

子模函数:
  f(C) = Σ_i max_{j∈C} S(i,j)

求解:
  Lazy Greedy, 贪心增长直到 |C| = 全量 10%
```

**关键约束**:
- 必须在候选池上运行（非全量数据），控制计算量
- 贪心迭代仅在候选池内采样，最终输出 = 全量 10%

### 2.5 阶段4: 梯度重要性精炼 (可选加分)

**对应脑机制**: 突触可塑性 — 大脑优先巩固"学不会"的样本

**算法**:
```
1. 用阶段3核心集训练 MLP
2. 在剩余数据上计算预测误差
3. 用高误差样本替换核心集中的低误差样本 (等量置换)
4. 迭代 2 轮
```

**约束**: 不做为核心流程，独立评估

## 三、实验设计

### 3.1 实验条件

| 编号 | 条件 | 数据量 | 说明 |
|------|------|--------|------|
| E0 | Full Data | 100% | 理论上界 |
| E1 | Random | 10% | 基线 (重复 3 次取平均) |
| E2 | Temporal-only | 10% | 仅阶段1 |
| E3 | Temporal + FL | 10% | 阶段1+3 (无 RAS) |
| E4 | **BRAIN-Coreset** | 10% | 阶段1+2+3 完整 |
| E5 | [加分] +Gradient | 10% | E4 + 阶段4 |

### 3.2 评估指标

- **主指标**: MSE (均方误差) — 7 自由度联合
- **辅指标**: MAE, 各维度单独 MSE (雷达图)
- **定性**: t-SNE 可视化, 动作分布直方图

### 3.3 数据划分

- 按 episode 划分: 40 train / 10 test
- 核心集选择在 train 集上进行
- 测试集固定，所有条件共用
